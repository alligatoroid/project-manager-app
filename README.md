# Project Manager App

A project management application.

## Directory Structure

```
src/           - Source code
docs/          - Documentation
examples/      - Example files
user-projects/ - User data
  ├── _inbox/    - Incoming items
  ├── _system/   - System files
  ├── _shared/   - Shared resources
  └── projects/  - User projects
```
